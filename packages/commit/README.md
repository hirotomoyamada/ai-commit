## Setup

1. Install

   ```sh
   pnpm add -g @hirotomoyamada/ai-commit
   ```

2. Retrieve your API key from [OpenAI](https://platform.openai.com/account/api-keys)

3. Set the key so `ai-commit` can use it

   ```sh
   ai-commit config set apiKey=<your token>
   ```
